# Методика анализа итогового CSV и пересчета выбросов

Вторая вкладка GUI работает с CSV, который был сохранен на первой вкладке.
Одна строка такого CSV соответствует одному стационарному режиму. При анализе
каждая строка обрабатывается отдельно: режимы не объединяются и не усредняются
между собой.

## Тепло режима

Для каждого режима берется колонка `Количество тепла, кДж`. Дополнительно
вкладка анализа рассчитывает тепло пара, тепло топлива и удельную теплоту
сгорания топлива:

```text
Qрежима, кДж = значение колонки "Количество тепла, кДж"
Qрежима, кВт·ч = Qрежима, кДж / 3600
Qтоплива, кДж = Qрежима, кДж - Qпара, кДж
```

`Qрежима, кВт·ч` выводится как справочная величина для отчета.
Методика расчета `Qпара` по разности энтальпии описана отдельно:
[METHODOLOGY_STEAM_ENTHALPY.md](METHODOLOGY_STEAM_ENTHALPY.md).

## Концентрации режима

Для каждого режима берутся уже рассчитанные на первой вкладке средние значения:

- `O2, %` из колонки `% O2, среднее`;
- `CO, ppm` из колонки `ппм CO, среднее`;
- `NO, ppm` из колонки `ппм NO, среднее`;
- `NO2, ppm` из колонки `ппм NO2, среднее`.

## Пересчет ppm в мг/кВт·ч

Пересчет выполняется по концентрациям конкретного режима и содержанию кислорода
этого же режима. Для CO используется коэффициент `1,25`, для NO и NO2 — `2,056`.
Поправочный множитель `10,83 / 11,38` задан методикой.

```text
CO, мг/кВт·ч = 1,25 · COppm · 21 / (21 - O2) · 10,83 / 11,38

NO, мг/кВт·ч = 2,056 · NOppm · 21 / (21 - O2) · 10,83 / 11,38

NO2, мг/кВт·ч = 2,056 · NO2ppm · 21 / (21 - O2) · 10,83 / 11,38
```

Здесь `O2` берется в процентах, `COppm`, `NOppm` и `NO2ppm` — в ppm. Для
корректного пересчета `O2` режима должен быть меньше `21 %`, иначе знаменатель
становится нулевым или отрицательным.

## CSV анализа

При сохранении результата второй вкладки создается CSV с разделителем `;` и
кодировкой UTF-8 с BOM. В нем одна строка соответствует одному режиму и
содержит:

- название эксперимента;
- название режима;
- начало и конец режима, если эти колонки есть в исходном CSV;
- количество тепла режима в `кДж` и `кВт·ч`;
- тепло пара, тепло топлива, массу топлива и удельную теплоту сгорания;
- `O2`, `CO`, `NO`, `NO2` для режима;
- пересчитанные `CO`, `NO`, `NO2` в `мг/кВт·ч`.
