# Методика расчета расхода воды по импульсам

## Основание

Для импульсного расходомера каждый полный импульс соответствует фиксированному
объему. Производители называют обратную величину K-фактором — числом импульсов
на единицу объема. Общая методика описана, например, в материале производителя
[Omega Engineering о K-факторе расходомера](https://www.omega.com/en-us/resources/flow-meter-k-factor).

Паспортное значение конкретного расходомера обязательно. По форме напряжения
его определить нельзя. GUI принимает `Vp` в литрах на импульс; если в паспорте
указано `K` имп/л, то `Vp = 1 / K`.

## Обнаружение импульсов

Предоставленный сигнал имеет два устойчивых уровня: около `−1,9 В` и
`+5,6…7,5 В`. На полном файле найдено 349 восходящих фронтов с типичным периодом
около 12,5 с.

Чтобы шум не создавал повторные импульсы, применяется триггер Шмитта:

1. Низкий и высокий уровни оцениваются как 10-й и 90-й процентили всех отсчетов.
2. Нижний порог равен `low + 0,35·(high − low)`.
3. Верхний порог равен `low + 0,65·(high − low)`.
4. Детектор сначала должен оказаться ниже нижнего порога (взведенное состояние).
5. Один импульс регистрируется при переходе выше верхнего порога.
6. Новый импульс невозможен до возврата ниже нижнего порога.

Так считается один восходящий фронт на физический импульс независимо от его
длительности и дребезга около одного порога.

## Формулы режима

Импульсы относятся к полуоткрытому интервалу `[начало, конец)`, чтобы фронт на
границе двух соседних режимов не учитывался дважды.

```text
N       = число восходящих фронтов
V, л    = N · Vp
Qv, л/мин = V · 60 / Δt, с
```

Для теплового расчета `Qv` считается постоянным средним расходом внутри режима:

```text
Qv, м³/с = Qv, л/мин / 60000
P(t), Вт = ρ · Qv · cp · (Tвыход(t) − Tвход(t))
Qтепло, Дж = интеграл P(t) dt методом трапеций
```

На коротком режиме возможна заметная дискретность: один дополнительный импульс
существенно меняет средний расход. Поэтому длительность режима желательно
выбирать существенно больше периода импульсов.
