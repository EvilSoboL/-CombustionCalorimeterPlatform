# Методика учета тепла пара

Во вкладке **Анализ результата** тепло, пришедшее с паром на распылитель,
вычитается из общего тепла режима. Оставшаяся часть считается теплом, полученным
из топлива.

## Исходные данные

Пользователь вручную задает:

- температуру на входе в парогенератор, `Tвх, °C`;
- температуру на выходе из парогенератора, `Tвых, °C`.

Из названия режима программа берет:

- расход топлива `F`, `кг/ч`;
- расход пара на распылитель `V`, `кг/ч`, если в имени режима есть часть
  `V...`;
- если распылитель задан как `A`, расход пара принимается равным нулю.

Например, для `F1V0.8A7.5` расход топлива равен `1 кг/ч`, расход пара равен
`0,8 кг/ч`, основной расход воздуха равен `7,5 кг/ч`.

## Разность энтальпии

Вкладка анализа оценивает разность энтальпии между водой на входе в
парогенератор и паром на выходе:

```text
Δh = hпара(Tвых) - hводы(Tвх)
```

Энтальпия воды на входе приближенно считается от `0 °C` по теплоемкости жидкой
воды:

```text
hводы(Tвх), кДж/кг = 4,186 · Tвх
```

Энтальпия пара на выходе считается как нагрев воды до `100 °C`, испарение при
`100 °C` и перегрев пара до температуры выхода:

```text
hпара(Tвых) = 4,186 · 100 + 2256,9 + hперегрева(Tвых)
```

Для участка перегрева от `100 °C` до `226,85 °C` используется приближенная
теплоемкость пара `2,08 кДж/(кг·°C)`. Для более высоких температур добавка
перегрева считается по уравнению Шомейта для H2O(g) из NIST Chemistry WebBook:

```text
t = T / 1000
H° - H°298.15 = A·t + B·t²/2 + C·t³/3 + D·t⁴/4 - E/t + F - H
```

Температура `T` подставляется в Кельвинах. Значение энтальпии получается в
`кДж/моль`, затем переводится в `кДж/кг` делением на молярную массу воды:

```text
M(H2O) = 0,01801528 кг/моль
Δhперегрева, кДж/кг = (H(T2) - H(T1)) / M(H2O)
```

Температура выхода парогенератора должна быть не ниже `100 °C`. Давление в этой
упрощенной модели явно не учитывается: вода на входе считается жидкой, пар на
выходе — газообразным. Для точного расчета при заданном давлении нужно заменить
эту оценку таблицами воды и пара или расчетом IAPWS-IF97.

Источник коэффициентов Шомейта: [NIST Chemistry WebBook, Water, Gas phase thermochemistry](https://webbook.nist.gov/cgi/cbook.cgi?ID=C7732185&Mask=1).

## Тепло пара

Масса пара за режим считается из расхода и длительности режима:

```text
mпара, кг = V, кг/ч · длительность, с / 3600
```

Тепло, внесенное паром:

```text
Qпара, кДж = Δh, кДж/кг · mпара, кг
```

Если в названии режима распылитель задан как воздух (`A`), то:

```text
mпара = 0
Qпара = 0
```

## Тепло топлива

Общее тепло режима берется из колонки `Количество тепла, кДж` итогового CSV.
Тепло, отнесенное к топливу:

```text
Qтоплива, кДж = Qобщее, кДж - Qпара, кДж
```

Масса топлива за режим:

```text
mтоплива, кг = F, кг/ч · длительность, с / 3600
```

Удельная теплота сгорания:

```text
qтоплива, кДж/кг = Qтоплива, кДж / mтоплива, кг
qтоплива, МДж/кг = qтоплива, кДж/кг / 1000
```
